# B站错误码说明

所有 B 站相关错误统一显示为：`RSS解析失败(-xx)`，其中 `xx` 为错误码。

## 应用内错误码
- 9001：网络不可用、超时、证书异常或请求抛出异常。
- 9002：缺少登录用户标识（DedeUserID）。
- 9003：默认收藏夹缺失或为空。
- 9004：二维码获取失败或解析失败。
- 9005：Cookie 无效或不完整。
- 9006：缺少播放参数（无法解析 `cid` 且没有可用的本地缓存）。
- 9007：播放地址为空。

## B站接口错误码
- 86038：二维码已过期。
- 其他正整数：B 站接口或 HTTP 返回的错误码，直接透传展示。
